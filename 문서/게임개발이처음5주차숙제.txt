# 내가 만들고 싶은 게임
- 게임 이름 : 하스스톤
- 게임에 대한 간단한 설명 : 유희왕 듀얼처럼 서로 카드로 싸우는 게임
- 게임 플레이 순서
1. 로비에서 30장을 갖춘 덱을 만든다.
2. 게임시작 버튼을 누른다.
3. 상대방과 매칭이 되면 게임을 시작한다.
4. 각 3장중 원하는 만큼 다시 덱에 섞어 넣은 후 그만큼에 카드를 수령 후 시작한다.
5. 자신에 차례가오면 카드한장을 뽑고 코스트(행동력)만큼 하수인을 소환하거나 마법카드를 사용한다.
6. 소환한 하수인이나 마법카드로 상대에 명치(목숨) 값을 0이하로 만들면 승리한다.
- 게임 내 구현해야 할 메커닉
선택한 카드들을 셔플
멀티동기화
